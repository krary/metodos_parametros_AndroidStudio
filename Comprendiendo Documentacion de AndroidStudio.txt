*****************
LAYOUTINFLATER
*****************


=====================================================================================================================================================================
La clase LayoutInflater en Android Studio es una clase que se utiliza para convertir un archivo XML de diseño en una vista o en un grupo de vistas correspondientes.
Se puede utilizar para cargar vistas en una actividad, fragmento o cualquier otro componente de la interfaz de usuario de una aplicación de Android.
Cuando se crea una interfaz de usuario en Android, la mayoría de los componentes se definen mediante archivos XML de diseño. La clase LayoutInflater permite a los 
desarrolladores de aplicaciones cargar estos archivos de diseño y crear objetos View que corresponden a las vistas definidas en los archivos XML.
La clase LayoutInflater es necesaria en muchos casos cuando se crea una aplicación de Android. Por ejemplo, se puede utilizar para crear vistas personalizadas, cargar
fragmentos dinámicamente, inflar menús de opciones y muchas otras tareas.Además, la clase LayoutInflater también es útil para crear diseños adaptables que puedan 
funcionar en diferentes tamaños de pantalla o dispositivos Android, ya que 
permite crear diferentes diseños para diferentes resoluciones de pantalla o orientaciones.En resumen, la clase LayoutInflater en Android Studio es una herramienta útil 
para cargar archivos de diseño de XML y crear vistas correspondientes en una aplicación de Android.
=======================================================================================================================================================================
